# Legacy API Service

REST API service simulating a legacy system for order management.

## Features
- Order listing
- Basic CRUD operations
- REST API endpoints

## Development
```bash
npm install
npm start
```

## Environment Variables
- `PORT` - Server port (default: 3001)
- `HOST` - Server host (default: 0.0.0.0)

## Architecture
- Express.js REST API
- Simple in-memory data storage
